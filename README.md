# Tennis Match Predictor

My project takes ATP data and uses it to predict two players and their suggested outcome.


How it works
- This is a heuristic-based prediction model, not a machine learning model. It uses manually selected weights applied to ATP match statistics to estimate player performance and simulate match outcomes by calculating a player's strength through their aces, double faults, serve percentage, and winner rank. It then uses a formula I picked and experimented with.

Features
- ATP match data analysis
- Serve percentage metrics
- Breakpoint conversion metrics
- Match winner prediction

Uses
- Python
- Pandas
